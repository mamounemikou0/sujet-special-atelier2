# Application de gestion d'utilisateurs (FastAPI + SQLite)

## Stack

- Backend : Python 3.11 + FastAPI
- Base de données : SQLite (`app.db`)
- Frontend : HTML + CSS + JavaScript vanilla
- Lancement : `uvicorn main:app --reload`
