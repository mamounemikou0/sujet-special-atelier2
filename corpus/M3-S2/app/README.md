# Corporate Room Booking System

Une application web d'entreprise classique (Multi-Page Architecture) pour la gestion et la réservation de salles de réunion.

## 🚀 Fonctionnalités
1. Inscription et Connexion persistante par session.
2. Système de réservation par créneaux fixes d'une heure (8h00 - 18h00) avec calendrier journalier adaptatif.
3. Tableau de bord pour le rôle "gestionnaire" (Taux d'occupation simulé, CRUD des salles, Annulation globale).
4. Détection automatique et notifications de rappels des réservations du jour.
