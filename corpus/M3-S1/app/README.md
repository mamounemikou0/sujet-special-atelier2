# Application de Gestion d'Utilisateurs

Application web complète avec FastAPI, SQLite et Vanilla JavaScript.

## Installation

1. Créer un environnement virtuel : `python -m venv venv`
2. Activer l'environnement : 
   - Windows : `venv\Scripts\activate`
   - Linux/Mac : `source venv/bin/activate`
3. Installer les dépendances : `pip install -r requirements.txt`

## Lancement

Exécuter la commande suivante à la racine du projet :
```bash
uvicorn main:app --reload
