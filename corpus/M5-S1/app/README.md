# Application de Gestion d'Utilisateurs

Application web de gestion d'utilisateurs avec FastAPI et SQLite.

## Fonctionnalités

- Inscription d'utilisateurs (email, nom d'utilisateur, mot de passe)
- Connexion/Déconnexion
- Page de profil modifiable
- Liste des utilisateurs
- Suppression de son propre compte
- Compte administrateur avec droits étendus
